# 🍽️ Zomato Clone – Responsive Food Delivery Website

A responsive front-end clone of the popular food discovery platform **Zomato**.  
This project recreates the modern UI design of the Zomato website including restaurant cards, offer badges, ratings, and responsive layout.

The goal of this project is to practice **front-end development, responsive design, and UI cloning of real-world platforms.**

## ✨ Features

✔ Responsive header with logo and navigation  
✔ Food category section (Pizza, Biryani, Burger, etc.)  
✔ Restaurant listing cards  
✔ Offer badges like **30% OFF**  
✔ Restaurant ratings  
✔ Delivery time information  
✔ Popular locality section  
✔ Explore options accordion section  
✔ Professional multi-column footer  
✔ Fully responsive (Mobile, Tablet, Desktop)
